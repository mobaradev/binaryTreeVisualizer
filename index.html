How to use

Getting started

Select the initial tree:

  • Start with an empty tree
  • Start with a random tree

Manage your tree

Tree can be managed from the side-panel on the left.

Insert the number into the input area.

This value will be used in functions available from the buttons below:

  • Insert
  • Delete (L)
  • Delete (R)
  • Find

Control the camera

Zoom

You can zoom in or out using your mouse wheel or touchpad gestures.

Alternatively, you can use Z key to zoom out and X key to zoom in.

Position

You can change the camera position using mouse of touchpad gestures. Hold left button and move cursor at the same time.

Alternatively, you can use WASD keys.

Observe tree's statistics

Statistics section provides numerical data that describes the tree.

Cells

Number of cells in the tree

Edges

Number of edges in the tree. This value is equal to number of cells - 1, or 0 - when the tree is empty.

Height

Height of the tree. The higher tree, the more operations have to be done to execute the functions.

Unique values

Number of unique values in the tree. Two or more of the same values in the tree are counted as one.

Functions

Insert

Inserts given value into the tree

Delete (L)

Deletes the value from the tree. If the cell has both children, the child from the left will be promoted.

Delete (R)

Deletes the value from the tree. If the cell has both children, the child from the right will be promoted.
